Step1 : Installation

Installation (Common Steps)

-Please, read the whole section, before attempting to install the theme.
-it is recommend to always backup your system before making any upgrades or changes!

  • There are two ways to install a SO Theme:

For installation of Opencart, you just need to follow these steps, Installation

Install Quickstart: By using this package, you will set the template exactly as our Demo with sample data.

Install Template : If you exist a opencart website you can install only template.

1.1 Install with Quickstart Package:

1.1.1 Create new database

Step 1: Open your Browser and type : localhost/phpmyadmin or yourdomain/phpmyadmin

Step 2: Then create database

1.1.2 Extract and Upload the whole quickstart package (admin, catalog, image … in tt_cygnus_quickstart folder ) to your server

1.1.3 Open browser and navigate to your site (where the quickstart uploaded) to load default Opencart installer with the database you just created.

1.1.4 Enter the username and password for the administration

1.1.5 Installation Complete

Note: At admin ,Go to > Extension > Modifications > Click on refresh button at top

System Requirements

  • These extensions must be enabled for OpenCart to install properly on the web server.
    • Web server: Web Server (Apache suggested)
    • PHP PHP 5.4+
    • MySQL Database (MySQLi suggested)

    Required PHP libraries / modules

    • Curl
    • ZIP
    • Zlib
    • GD Library
    • Mcrypt
    • Mbstrings
    • Xml

  • To install this theme you must have a working version of opencart already installed. For information in regard to installing the opencart platform, please see the opencart Wiki :

    http://docs.opencart.com/en-gb/getting-started

1.2 Install with Template Package:

Upload Theme Files

After you complete the installation of opencart, please follow below steps to install themes

  • Step 1 - Unzip the Greenesta.zip package.
Replace Theme Files
  • Paste these directories from 'Theme-package' to your opencart root folder.
  • "admin" (Overwrite this folder)
  • "catalog" (Overwrite this folder)
  • "image" (Overwrite this folder)

AFTER UPLOAD THEME FILES, YOU MAY GET ERROR(COULD NOT LOAD FILE) ON FRONT SIDE.
BUT DON'T WORRY, WHENYOU ACTIVE OUR THEME, IT IS WORK PERFECTLY.

After Completing Installation Steps Must be Select Theme Store In Admin Panel for that:
  • For Default Home : Go to Extensions > Extensions > Themes > Greenesta > Edit

Step2 : Logo and Image size settings

  • For Logo setting , Go to System > Setting > Edit > Images,
  • From Store Logo you can upload your logo image.
For Image Size setting ,
For Default_Home : Go Extensions > Extensions > Themes > Greenesta > Edit ,
  • Change category image thumb size to: 200 X 180,
  • Change product image thumb size to: 620 X 620,
  • Change Product Image Popup Size to: 700 X 700,
  • Change product image list size to: 283 X 283 ,
  • Change additional product image size to: 140 X 140,
  • Change related product image size to: 283 X 283 ,
  • Change wish list image size to: 90 X 90 ,
  • Change cart image size to: 100 X 100 ,
  • Change store image size to: 268 X 50 .
  • Click on “Save” button.
  • For Store setting Go To System->Settings Click on store Edit link, and set below settings.

Step 3 : Language Settings

To Enable Laguages login to your admin panel and do follwing settings.

  • For Example To add Arabic Language :
  • Go to System > Localisation > Languages,
  • Click on “Add New” button to add new Language.
  • Language Name : Arabic,
  • Code : ar,
  • Locale : ar-UN,ar_UN,UTF-8,Arabic,
  • Status : Enable,
  • Sort Order : as you like.
  • Click on “Save” button.
